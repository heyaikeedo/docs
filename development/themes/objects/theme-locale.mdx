The `theme_locale` variable provides information about the current language and region settings in Aikeedo themes.

## Availability

The `theme_locale` variable is available globally throughout the theme.

<ResponseField name="theme_locale" type="string" required>
  The current theme locale value. This represents the language and region settings, following the format of language code and country code (e.g., "en-US").
</ResponseField>
